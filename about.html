About Kyle Tech Corner

Operated by Kyle Cosper · Temple, Texas, USA

Kyle Tech Corner is a personal tech project run by Kyle Cosper, a technology enthusiast based in Temple, Texas. It covers my home lab, networking, automation, and DIY tech builds — shared on the @KyleTechCorner YouTube channel.

The home lab

At the center of it all is a self-hosted home lab: a small set of servers and network gear I run at home to host services, experiment with new tech, and automate everyday tasks. Like any real infrastructure, it needs monitoring — which is why I built a notification system that texts me when something needs attention.

The notification system

The lab runs a self-hosted application that sends automated SMS alerts to me, the operator — covering system status, monitoring thresholds, environmental/weather monitoring, and backup reports. Full details, including how opt-in and opt-out work, are on the Notifications page.

Who runs this

Everything here — the channel, the website, and the home lab — is built and maintained by one person, Kyle Cosper. This is a personal, non-commercial project. Questions are welcome at kyleecosper@gmail.com.
