Kyle Cosper Home Lab — SMS Notification Opt-In

This page documents how the operator of the Kyle Cosper Home Lab consents to receive automated SMS notifications (server status alerts, monitoring notifications, and backup reports) from his own self-hosted monitoring system.

Opt-In Workflow

Opt-in occurs inside the self-hosted Home Lab application, on the Settings → Notifications page. The operator enters his own mobile phone number and must check the consent checkbox below before SMS notifications are enabled. No third parties are ever messaged; the operator is the sole recipient.

Settings → Notifications → SMS Alerts

I consent to receive automated SMS notifications from Kyle Cosper Home Lab at the number provided. Message frequency varies. Message & data rates may apply. Reply STOP to opt out, HELP for help. See our Privacy Policy.

(Illustration of the in-app opt-in screen. The "Enable SMS Alerts" button is disabled until the consent box is checked.)

Message Types

  • Server status alerts (e.g. host back online, service down)
  • Resource/monitoring notifications (CPU, memory thresholds)
  • Scheduled backup completion reports

Opt-Out

Reply STOP to any message to unsubscribe at any time. Reply HELP for assistance. Contact: kyleecosper@gmail.com.
